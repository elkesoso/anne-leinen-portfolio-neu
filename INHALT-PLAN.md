# Inhaltsplan — Anne Leinen Portfolio
> Erstellt: 29.03.2026
> Ziel: Alle Inhalte vorbereiten BEVOR Kadence gebaut wird

---

## Warum erst Inhalte, dann bauen?
Wenn du zuerst die Inhalte hast, kannst du in Kadence direkt echte Texte und Bilder einfügen
statt Platzhalter. Das spart die doppelte Arbeit und du siehst sofort wie es wirklich aussieht.

---

## Phase 1 — Ordnerstruktur anlegen
**Aufwand:** 10 Min | Einmalig

Erstelle diesen Ordner auf deinem Mac:

```
~/Downloads/anne-leinen-INHALTE/
├── bilder/
│   ├── gemaelde/          ← alle 20 Gemälde-Fotos
│   ├── startseite/        ← die 4 ausgewählten Highlight-Bilder
│   ├── portrait/          ← Foto von Anne Leinen
│   ├── atelier/           ← Atelier-Fotos (falls vorhanden)
│   └── mockups/           ← Kunstwerk-an-der-Wand-Fotos
└── texte/
    ├── ueber-mich.txt
    ├── gemaelde-liste.csv
    ├── ausstellungen.txt
    └── kontakt.txt
```

---

## Phase 2 — Bilder sammeln & benennen
**Aufwand:** 1–2 Std

### Gemälde (20 Stück für Kunstwerke-Seite)
Jedes Bild erhält einen einheitlichen Dateinamen:

```
NR_titel-kurz_jahr.jpg
Beispiel: 01_aurelia-I_2024.jpg
```

Für jedes Bild notierst du in `gemaelde-liste.csv`:

| Nr | Dateiname | Titel DE | Titel EN | Technik DE | Technik EN | Breite cm | Höhe cm | Jahr | Status |
|----|-----------|----------|----------|------------|------------|-----------|---------|------|--------|
| 01 | 01_aurelia-I_2024.jpg | Aurelia I | Aurelia I | Mischtechnik auf Leinwand | Mixed Media on Canvas | 120 | 120 | 2024 | Auf Anfrage |
| 02 | ... | | | | | | | | |

**Status-Optionen:** `Verfügbar` / `Auf Anfrage` / `Privatsammlung` / `Verkauft`

### Startseite (4 Highlight-Bilder)
- Wähle 4 der 20 Gemälde aus, die auf der Startseite erscheinen
- Kopiere sie zusätzlich in den Ordner `startseite/`
- Notiere welche 4 es sind (Dateinamen)

### Portrait
- Handy-Foto vorerst ok → in `portrait/` ablegen
- Dateiname: `anne-leinen-portrait.jpg`
- Professionelles Foto später einfach ersetzen (gleicher Dateiname!)

### Mockups
- Fotos mit Kunstwerk an echter Wand → in `mockups/`
- Dateinamen: `mockup-01.jpg`, `mockup-02.jpg` usw.

---

## Phase 3 — Bilder optimieren
**Aufwand:** 30 Min (kann ich für dich automatisieren!)

Für die Webseite brauchen Bilder die richtige Größe:
- Gemälde: max. 2000px breit, unter 500 KB
- Portrait: max. 1200px breit
- Mockups: max. 2000px breit

**Handy-Fotos sind oft 4000px+ und 5–10 MB** → zu groß für Web, Seite lädt langsam.
→ Sage Bescheid wenn alle Bilder gesammelt sind, dann optimiere ich sie per Script.

---

## Phase 4 — Texte schreiben
**Aufwand:** 2–4 Std (je nach vorhandenem Material)

### 4.1 Über mich (`ueber-mich.txt`)
Schreibe zwei kurze Absätze auf Deutsch, dann auf Englisch.
Vorlage (kannst du anpassen):

```
[DE]
Haupttext (2–3 Sätze, wer ich bin und was mich antreibt):
...

Ateliertext (1–2 Sätze, wie ich arbeite / meine aktuelle Serie):
...

[EN]
Main text:
...

Studio text:
...
```

### 4.2 Kontaktdaten (`kontakt.txt`)
```
E-Mail:
Instagram-Handle: @...
Instagram-URL: https://www.instagram.com/...
Ort / Stadt:
Atelierbesuche: Nach Vereinbarung / Nein / ...
```

### 4.3 Ausstellungen (`ausstellungen.txt`)
Für jede Ausstellung:
```
Titel:
Typ: Solo / Gruppe / Vernissage / Finissage
Location:
Stadt, Land:
Datum (von – bis):
Status: Aktuell / Kommend / Vergangen
Kurzer Beschreibungstext (1–2 Sätze, DE):
Kurzer Beschreibungstext (1–2 Sätze, EN):
RSVP-Link (falls vorhanden):
```

### 4.4 Gemälde-Beschreibungen (optional aber empfohlen)
Für jedes der 20 Gemälde optional eine kurze Beschreibung (für Lightbox / Detailseite):
```
Titel: Aurelia I
Beschreibung DE: Gold steht hier nicht für Prunk, sondern...
Beschreibung EN: Gold here does not represent opulence, but...
```

## Phase 5 — Texte übersetzen
**Aufwand:** 1–2 Std (oder per KI schneller)

Alle deutschen Texte müssen ins Englische übersetzt werden.
→ Kann ich für dich übernehmen, sobald die deutschen Texte fertig sind!

## Phase 6 — Inhalte prüfen (Checkliste)

Bevor Kadence gebaut wird, sollte folgendes abgehakt sein:

**Bilder:**
- [ ] 20 Gemälde-Fotos vorhanden und benannt (01_... bis 20_...)
- [ ] 4 Startseiten-Highlights ausgewählt
- [ ] Portrait-Foto vorhanden
- [ ] Mindestens 1 Mockup-Foto vorhanden
- [ ] Alle Bilder optimiert (max. 2000px, unter 500 KB)

**Texte:**
- [ ] Über-mich-Text DE fertig
- [ ] Über-mich-Text EN fertig
- [ ] Kontaktdaten vollständig
- [ ] Mindestens 3 Ausstellungen dokumentiert
- [ ] `gemaelde-liste.csv` vollständig ausgefüllt (alle 20 Werke)
- [ ] Gemälde-Beschreibungen (optional)

**Technisch:**
- [ ] Alle Dateien im Ordner `anne-leinen-INHALTE/`
- [ ] Einheitliche Dateinamen (keine Leerzeichen, keine Umlaute!)
